To do list

Continue your to do list, so that you can have multiple lists, each with its own set of tasks.

There are two approaches to the form for adding new tasks to a list:

  1. When a list's name is clicked, remove the form for the previously-displayed list from the DOM, and insert a new form to add a task to the current list.
  2. Code the form into HTML. Each time that a list's name is clicked, use the jQuery off method to remove any existing event listener. Then bind a new event listener that causes the form to create a task and add it to the newly-selected list.
